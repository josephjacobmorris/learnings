# Good Advices

- Observe, but don't absorb.

- Love, but don't depend.

- Want, but don't need.

- Feel, but don't dwell.

- Give, but don't expect.

- Speak, but don't hurt.

## On Learning

- Learn daily
- Learn Basics first
